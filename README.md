## 프로젝트 소개
이 프로젝트는 JWT 기반 인증과 Spring Security를 활용한 보안 기능을 구현하는 예제입니다. 주요 목표는 필터 기반 인증/인가 로직을 구현하고 JWT 발행과 검증을 위한 테스트를 수행하는 것입니다.

## 주요 기능
- Spring Security를 이용한 필터 기반 인증/인가
- JWT 발행 및 검증 로직 구현
- Access/Refresh Token 관리
- JUnit을 활용한 유닛 테스트
- EC2에 애플리케이션 배포
